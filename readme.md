## 📌 Project Overview

Todo Manager is a web application that allows users to create personal accounts and manage their daily tasks in an organized way.

Users can register, log in securely, and perform all CRUD operations (Create, Read, Update, and Delete) on their tasks. After successful registration, an email notification is automatically sent to the user's registered email address.

This project demonstrates the implementation of Django Authentication, CRUD Operations, Email Integration, Template Rendering, and Database Management.

## ⚙️ Installation

### Clone the Repository

```bash
git clone <repository-url>
cd todo-manager
```

### Create Virtual Environment

```bash
python -m venv venv
```

### Activate Virtual Environment

```bash
venv\Scripts\activate
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Apply Migrations

```bash
python manage.py migrate
```

### Run the Development Server

```bash
python manage.py runserver
```

Open your browser and visit:

```text
http://127.0.0.1:8000/
```
